TDengine Cloud 在 2023年11月01日发布新版本**20231101**。请查看下面的详情

## 功能

1. 在**工具**增加新的工具 Seeq 并且增强 Grafana 的内容
2. 在**用户设置**页面增加操作的 MFA 的设置
3. 在**数据写入**增强用户体验
4. 在**通知**页面增加批量操作

## 修复问题

1. 通过 REST API 访问系统的 SQL 查询接口可能导致系统组件内存溢出
2. 通过Websocket压缩算法 permessage-deflate 无法建立和网关的连接
